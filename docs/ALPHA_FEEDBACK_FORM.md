# Alpha Feedback Form — content for Google Forms / Typeform

Recreate this in your form tool of choice and share the link in `#ai-gov-alpha`.

## Section 1 — About you (one time)

1. **Your name** (text, required)
2. **Your team** (Engineering / Sales / Marketing / Operations / Other) (select)
3. **Primary OS** (Windows / macOS / Linux) (select)
4. **Primary browser** (Chrome / Edge / Brave / Firefox / Safari) (select)

## Section 2 — Daily ping (repeat as needed)

1. **Today's date** (date)
2. **Did the agent run today?** (Yes / No / Don't know)
3. **Did the browser extension behave normally?** (Yes / No issues / Saw a problem)
4. **Did you notice any slowdown or unusual CPU usage?** (Yes / No)
5. **Did you see any in-page warnings from the extension?** (Yes / No)
   - If yes, was the warning correct? (Yes / False positive / Not sure)
6. **Did you encounter any AI tools today that the dashboard didn't list?** (open text)

## Section 3 — Trust & comfort

1. **How comfortable are you with this monitoring?** (1–5 Likert, 1 = uncomfortable, 5 = fine)
2. **Anything you wish the agent did NOT collect?** (open text)
3. **Anything you wish it DID collect?** (open text)
4. **Did you re-read the employee disclosure? Did anything surprise you?** (open text)

## Section 4 — Bugs / regressions

1. **Describe the bug** (open text)
2. **OS + version** (text)
3. **Reproduction steps** (open text)
4. **Severity** (Cosmetic / Annoying / Blocking work)

## Section 5 — Quitting

1. **Are you leaving the alpha?** (Yes / No)
2. **Why?** (open text)
3. **Did you uninstall? Did it work cleanly?** (Yes / No / N/A)
