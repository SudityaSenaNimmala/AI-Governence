Browser Extension

The CloudFuze browser extension intercepts sensitive data in web-based AI services (ChatGPT, Claude.ai, Gemini, and more). It scans prompts and file uploads before they leave the browser and reports violations to the governance server.

Installation Guide

1

Download the extension

Click below to save the extension folder to your computer. You'll pick where to save it.

2

Open Chrome Extensions page

Navigate to chrome://extensions in your Chrome or Edge browser.

Or click: Open Chrome Extensions

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page.

4

Load the extension

Click "Load unpacked" and select the CloudFuze-Extension folder you downloaded in step 1.

5

Configure the extension

Click the CloudFuze extension icon in the toolbar and set the server URL to match your governance server (default: http://localhost:8787).

6

Verify it's working

Visit ChatGPT or Claude.ai. You should see the CloudFuze shield icon indicating the extension is active. Try pasting a test secret like AKIA1234567890ABCDEF to trigger a detection.

Desktop App Hooks

Requires Administrator. ASAR injection modifies the Electron app bundle of AI desktop apps (Cursor, older Claude Desktop builds). Apps that enforce ASAR integrity (current Claude Desktop) will break — use the OS monitor + proxy for those instead.

Inject DLP hooks into desktop AI apps

This scans for installed Electron-based AI apps and injects a DOM-level hook that intercepts sensitive data before it's sent. The hook reports violations to the governance server.

Supported apps: Cursor, Claude Desktop (non-Store builds), ChatGPT Desktop, GitHub Copilot.


        

      


      

        
Alternative: OS Monitor + Proxy

        
For apps that enforce ASAR integrity, the OS-level monitor (clipboard, typed prompts, file dialogs)
           and the HTTPS DLP proxy provide full coverage without modifying the app bundle.

        
    
          
  • OS Monitor — detects sensitive content pasted or typed into any AI app (active when monitoring is running)
    • 
          
  • HTTPS Proxy — network-level interception that blocks requests containing secrets (run via CLI: ai-gov-agent --proxy)
